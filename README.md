# one-for-all

---

## Usage

```javascript
~$ one-for-all -h
Usage: one-for-all [options]

A CLI for generating NestJS or Rust projects

Options:
  -v, --version            version of the cli
  -t, --template <engine>  specify template engine (default: "nestjs")
  -d, --deploy <system>    specify template engine (default: "terraform")
  -n, --name <name>        specify a project name
  -h, --help               display help for command
```
